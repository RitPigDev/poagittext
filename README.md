# poagitText

poagitText is a text editor built fully into a single HTML file. It's designed to run on old/slow computers while having a decent feature set.
<b>You need to have a reasonably modern browser to use this.</b>

While it may not seem like it, I promise that I'm working on the next update. I only work on poagitText in my free time, so it takes me a while to code something that won't break everything.
